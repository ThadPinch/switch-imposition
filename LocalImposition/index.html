Local Imposition

Pick a PDF, choose the imposition, and download the imposed file.

This page is backed by a local Python server so very large PDFs stay off the browser heap. It mirrors the layout and page-order behavior from batchimposition.ts: repeat mode fills every position with the current artwork page, and cut-and-stack uses position-major page ordering.
